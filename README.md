# AsturiasOculta

## Descripción

<b>Asturias Oculta</b> consiste en una aplicación web interactiva centrada en dar a conocer lugares pocos conocidos del Principado de Asturias.

Este proyecto busca crear un encuentro digital donde los usuarios puedan descubrir, explorar y compartir localizaciones únicas, con el fin de incentivar un turismo más participativo y auténtico, fomentando el turismo local así como sus costumbres y tradiciones.

## Tecnologías utilizadas

- HTML5 + CSS3
- Bootstrap
- Leaflet.js
